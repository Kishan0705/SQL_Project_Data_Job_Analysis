## üîç Question 2: Identify the companies that have posted job listings requiring all top 5 skills for "Data Analyst" positions, while offering health insurance and work-from-home options.

```python
import numpy as np
import pandas as pd
import matplotlib.pyplot as plt

# Load the data from CSV
file = r"path"
df = pd.read_csv(file)

# Assuming 'company_name' and 'jobs_requiring_all_skills' are the column names
df['jobs_requiring_all_skills'] = df['jobs_requiring_all_skills'].astype(int)

# Create the figure with a dark background
plt.figure(figsize=(10, 6))
plt.style.use('dark_background')

# Normalize the range for the color map
colors = plt.cm.Blues(np.linspace(0.3,1, len(df)))

# Plot the horizontal bar chart with the gradient colors
df.plot(kind='barh', x='company_name', y='jobs_requiring_all_skills', color=colors, edgecolor='white', legend=False)

# Set the title and axis labels with appropriate font sizes and color
plt.title('Top Companies Requiring All Skills for Data Analyst Roles', fontsize=16, color='white', pad=20)
plt.xlabel('Number of Jobs Requiring All Skills', fontsize=12, color='white', labelpad=10)

# Invert y-axis to show the most frequent company at the top
plt.gca().invert_yaxis()

# Add light gridlines for readability
plt.grid(visible=True, axis='x', color='gray', linestyle='--', linewidth=0.5)

# Improve layout and appearance
plt.tight_layout()

# Display the chart
plt.show()
```
